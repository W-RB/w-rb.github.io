【有惊无险】CSP-J初赛游记

发表于 2020-11-05 更新于 2020-11-04 分类于 随笔

也许应该要从10.11说起？

念念不忘，必有回响

下午2:30考试，我1:00左右到了lyyz。今年初赛居然就在我们学校考。来到之后发现只有DAZ，去找小熊量了个体温。

有点慌，因为我基本就没怎么学初赛。就抱着“能进就进”的态度去考了试。2:00左右，Jair，椋枨，LMT，wyy，zzy等几个一块来了，场面好不壮观。原来我进来之后他们都在校门口等着的？

去机房玩了一会儿，有幸遇到两位提高组dalao。不久，就下去了机房。我在102考场，跟wyy和zjy两位一起。Jair，DAZ，LMT和zzy好像都在二楼。

拿下试卷，选择题都很简单。差不多十多分钟，选择题就写完了。

但是！但是！但是！后面的阅读程序是真的让我蒙了！

Like this：

#include <algorithm>
#include <iostream>
using namespace std;
int n;
int d[50][2];
int ans;
void dfs(int n, int sum) {
    if (n=-1) {
        ans = max(sum,ans);
        return;
    }
    for (int i = 1;i < n; ++i) {
    int a = d[i -1][0]，b= d[i - 1][1];
    int x = d[i][e], y = d[1][1];
    d[i- 1][0] = a + x;
    d[i-1][1] = b + y;
    for(int j = 1; j < n - 1; ++j)
        d[j][0] = d[j + 1][0], d[j][1] - d[j + 1][1];
    int s = a + x + abs(b - y);
    dfs(n - 1, sum + s);
    for (int j = n-1;j > 1; --1)
        d[j][0] = d[j - 1][0], d[j][1] = d[j - 1][1];
    d[i - 1][0] = a, d[i - 1][1] = b;
    d[i][0] = x，d[i][1] = y;
int main() {
    cin >> n;
    for(int i = 0;i < n;++i)
        cin >> d[1][0];
    for(int i=0;i<n;++1)
        cin >> d[1][1];
    ans = 0;
    dfs(n, 0);
    cout << ans << endl;
    return 0;
}

假设输入的n是不超过50的正整数，d[1][θ]、 d[i][1]都是不超过10000的正整数，完成下面的判断题和单选题:

●判断题
1) 若输入n为0，此程序可能会死循环或发生运行错误。( )

2) 若输入n为20，接下来的输入全为日。则输出为0。( )

3) 输出的数一定不小于输入的d[1][0]和d[1][1]的任意一个。( )

●单选题
4) 若输入的n为20，接下来的输入是20个9和20个e，则输出为
()
A.1917  B.1908  C.1881  D.1890

5) 若输入的n为30，接下来的输入是30个日和30个5，则输出为
()。
A.2020  B.2030  C.2010  D.2000

6)  (4分) 若输入的n为15，接下来的输入是15到1。以及15到1.则输出为()。
A.2420  B.2220  C.2440  D.2240

看完这个题当时我脑子就炸了。这是什么鬼玩意啊

不过所幸，后面的完善程序还算简单。

2 HOURS LATER

走出考场，心里大喊：废了废了。看到同学们都这样的模样。Jair说他废了（然鹅他70+，这个骗子），椋枨说他还好。wyy和DAZ都是一副面对垃圾试题才会露出的表情。

然后回到了机房，倾听了ljq学长的至理名言：

这小小普及组，不考也罢！

咕了个分，62.5，有点慌。然后老师给我说我59分，而且卡在几个线上。ly一共30个名额，30，31，32名都是59分。心里顿时有点慌了。

没想到明一早，老师说我过初赛了！顿时心里的一块大石头落了地。Jair，椋枨，LMT好像都70+，zzy59.5，DAZ49没过，有点可惜。wyy也没过……

初赛就这样，有惊无险。接下来，要准备好参加复赛咯~
